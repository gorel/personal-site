    return x % 2 == 1

my_list = [1, 2, 3, 4, 5]

print(new_list)

  • [overlap] Write a program that takes two filenames on the command line. The program should output all lines that occur in both files. For example:
# file: a.txt
This is line 1.
This is another line.
Here is a third line!
I love Python!

# file: b.txt
I am another file
I love Python!
This is another line.

# running our program
$ python3 overlap.py a.txt b.txt
This is another line.
I love Python!

What's next?

Congratulations! You've now finished the content of the level 2 course. You should have a solid understanding (or at least familiarity) with all of the major tools and concepts used as the building blocks for programming. Depending on your interests, there are many different ways you could go from here to extend your learning.

I want to learn more Python specifically

  • Named parameters and default values
  • Functions as parameters
  • Exceptions and exception handling
  • Lambda functions
  • Decorators

I want to learn more computer science

  • Data structures (sets, stacks, trees, maps)
  • Inheritance
  • Recursion
  • Regular expressions (regex)

I want to learn some specific libraries for practical use

  • numpy (scientific computing / data science)
  • scipy (mathematics, science, engineering, data science)
  • wxPython (desktop applications)
  • tkinter (applications with a graphical component -- not just terminal-based)
  • flask or django (web development)
  • nltk (natural language toolkit -- parsing and working with human language)
  • scrapy (scraping websites -- ex, doing something fun with wikipedia links)
  • sklearn (machine learning)

I want to take online courses to reinforce my learning

  • MIT OCW - assumes no programming background
  • edx - Slightly faster-paced than the MIT course
  • Coursera course for python beginners
  • Udacity - Focuses on object-oriented programming
  • Coursera - Principles of Computing Part 1
  • Coursera - Principles of Computing Part 2

I just want to practice writing more projects

  • write a "magic 8 ball" program
  • create a text adventure game
  • write a program for a simple game (ex, rock-paper-scissors or checkers) and create a "bot" program to play against you
  • write a program that connects to your gmail account and reads your emails (libgmail)
  • write a program that connects to your twitter account and reads posts (python-twitter)
  • write a program that sends sms messages (twilio)
  • write a program that download web pages once a day and alerts you if the page changes (urllib2)
  • write a program that downloads a random wikipedia article (urllib2)
  • write a program that gives you all the words you can make from a scrabble set of letters
  • write a program that downloads all the facebook posts on a user's timeline (python-facebook)
