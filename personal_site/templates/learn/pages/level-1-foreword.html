Programming Beginners: Introduction

Ever wanted to learn to program but not sure where to begin? This class will help guide you through some topics you might learn about in a first semester CS course.

No prior programming experience required or expected.

Setup

At a minimum, you should install Python 3 and a text editor. If you're using a mac, there's a good chance you have these by default. Open a terminal (search for Terminal in finder) and type python3. If you don't get an error message, you have Python 3! For a text editor, I personally enjoy Atom, but there are many different options.

If you're on Windows, you will likely have to install these yourself. Check out the link above to install Python 3 and for a text editor, I highly recommend Notepad++, but again, there are many viable options.

1. Install a text editor

You only need to install one of these. I personally recommend Atom, but any one of them will work fine. Just take a look and choose the one you think looks best.

For Mac: Atom, Sublime Text 2 or 3, TextMate

For Windows: Notepad++, Crimson Editor, Sublime Text 2 or 3, Atom

2. Read this command line tutorial

Read this short tutorial: https://tutorial.djangogirls.org/en/intro_to_command_line/

Please note that even if you're on Windows, you should follow the instructions for OS X and Linux since you will be using powershell.

Again, if you're on Windows, you should follow the instructions for OS X and Linux since you will be using powershell.

For Mac: iTerm or Terminal (both already installed by default)

For Windows: powershell (already installed by default)

From now on, we will call the command line window "the terminal".

3. Install Python 3

There's a small chance that Python 3 may already installed on your machine. To test this, open Terminal on Mac (or powershell on Windows) and run python3. If you see a message like "python3: command not found" you do not have it installed.

If it is not installed already, navigate to https://www.python.org/downloads/ and follow the instructions to install the latest version of Python 3 on your machine.

IMPORTANT: You will not be able to perform the exercises without this setup!
