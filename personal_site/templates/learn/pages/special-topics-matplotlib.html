The matplotlib library

Setup

Before beginning this lesson, you will need to install and set up Jupyter notebook. Instructions for this are listed here, though they may vary slightly between systems.

To begin, open a terminal and run the following commands ($ represents the terminal prompt)

$ pip3 install --user numpy pandas matplotlib jupyter
( you'll see lots of install messages here )
$ jupyter-notebook
( a new window should open in your browser like http://localhost:8888/tree )

In the top right of the web page that opened, click New → Python3 notebook. Type the following in the first cell of the notebook:

import pandas as pd
import numpy as np
import matplotlib.pyplot as plt
print("Success!")

Hit the Run button in the toolbar. If you didn't get an error, great! You're all set for class. If you did get an error, send it to me, and I'll help you debug what's going wrong.

From here, you're ready to go through the interactive tutorial below. You may wish to play around in Jupyter first to get accustomed to how it works. It's extremely powerful and there are many features you may enjoy.
