Exercises

  • [vowels] Write a function called is_vowel(c) that takes a character (a string of length 1) and returns True if the character is a vowel (recall that vowels are A, E, I, O, and U).
  • [divisors] Write a program that takes three ints on the command line (call them lower upper and divisor) then prints out all numbers between lower and upper that are divisible by divisor. For example:
$ python3 divisors.py 1 100 11
11 22 33 44 55 66 77 88 99
  • [letterfreq] Write a program that takes a filename on the command line and prints out how often each letter occurs in the file text. For example, if the file text was the quick brown dog jumps over the lazy dog, your output would look like this:
